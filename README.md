Scouter Resources


A mobile-friendly prototype built for the Scouting Ireland Safety Advisory Team (SAT) review
A potential solution to for giving scouters easy and  quick access to safety guidance, a pre-meeting checklist, and a link to the official incident reporting form.

Status

This is a prototype for SAT review, shared for discussion and feedback. It is not an official Scouting Ireland resource, has not been approved for rollout, and should not be treated as a live channel or relied upon in place of official Scouting Ireland guidance.
